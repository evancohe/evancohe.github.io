Octopath Traveler - 八方旅人

EvanCohe

Game › Nintendo
Game, Nintendo

最近在玩的游戏: Octopath Traveler - 八方旅人

从学者开始,前期开荒还是很轻松的,基本用属性克制可以一路找齐其他7人..

2.5d / 像素 / 移轴 的画面渲染很好看

我藏起来了.

两天后的现在又感觉…其实八方旅人的剧情是单人的,也就是说无论你从哪个角色开始,多少个角色组队,到了剧情位置也只是剧情主角一人展开,和旅人没什么交集,而且互动对话这个机制(每段主线剧情过后,会触发队伍里两个角色的几段对话)…让人感觉是强塞进去加强剧情互动感的…其实这游戏应该叫 1+1+1+1+1+1+1+1旅人.

所以正确的理解是 买画面,送游戏 :P
